# Account-Restful-Api
## Dependencies:
* flask
* flask_sqlalchemy
* flask_marshmallow
* os
* json
* werkzeug.security
* uuid
* jwt
* datetime
* functools
* configparser

## Installation:
Repo:
```git
git clone https://github.com/lubek-dc/Account-Restful-Api
```

Libs:
```py
pip install flask flask_sqlalchemy flask_marshmallow os json werkzeug.security uuid jwt datetime functools configparser
```
